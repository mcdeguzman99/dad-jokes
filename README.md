# Dad Jokes

## Description

This demo app shows ten random Dad Jokes.

## Getting Started

### Installing

- Clone this repository
- `npm i`
- `npx webpack`

### Dad Jokes

To see ten random Dad Jokes, view http://127.0.0.1:3000/dist/index.html in your browser.
